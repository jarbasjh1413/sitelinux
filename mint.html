Linux Mint

O objetivo do Linux Mint é produzir um sistema operacional moderno, elegante e confortável que seja poderoso e fácil de usar. Linux Mint é uma das distribuições Linux para desktop mais populares e usada por milhões de pessoas.

Algumas das razões para o sucesso do Linux Mint são:

  • Funciona fora da caixa, com suporte completo de multimídia e é extremamente fácil de usar.
  • É gratuito e de código aberto.
  • É voltado para a comunidade. Os usuários são encorajados a enviar feedback sobre o projeto para que suas idéias possam ser usadas para melhorar o Linux Mint.
  • Baseado em Debian e Ubuntu, oferece cerca de 30.000 pacotes e um dos melhores gerenciadores de software.
  • É seguro e confiável. Graças a uma abordagem conservadora para atualizações de software, um gerenciador de atualizações exclusivo e a robustez de sua arquitetura Linux, Linux Mint requer muito pouca manutenção (sem regressões, sem antivírus, sem anti-spyware ... etc).
